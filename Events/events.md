# Event Module

Um mecanismo para:
    [] Disparar eventos
    [] Ouvir eventos
    [] Fazer alguma ação quando ouvir o evento
    [] É a base para muitos outros módulos como: http, stream, file system, etc...

## O que vamos ver:
    [] Utilizar os eventos
    [] Disparar eventos
    [] Ouvir eventos
    [] Executar ações para determinados eventos
    [] Entender como ele é a base para outros módulos
